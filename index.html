個人簡介

您好，我是陳雅玟，畢業於國立臺北商業大學數位多媒體設計系，具備實作導向的學習背景與跨領域專案經驗，除了參與多項 VR 教材與互動展覽開發，我也協助過系上教室主機與區域網路的日常維護，逐漸對系統與網路技術產生濃厚興趣。畢業後，我參加緯育 TibaMe 的「資安與雲端架構工程師養成班」，開始深入學習雲端平台、系統維運與網路管理。

在訓練期間，我操作過 AWS EC2、VPC、RDS、IAM、IAM Identity Center、CloudWatch、App Runner 等服務，透過 ShellScript 撰寫自動化腳本與 Dockerfile 封裝應用，再推送至 ECR 並使用 App Runner 完成無伺服器部署。此外，我也實作過 Nginx 的反向代理與負載平衡設定，並完成 ELB 的基本環境架設與測試。

在專題《星探》中，我整合 AWS Bedrock（Claude 模型）與 Dify Workflow，設計提示詞與回應流程，實作讓使用者可透過自然語言查詢 EC2 狀態、帳單與 CloudWatch Logs 的功能，讓模型能依語意自動觸發 API 並整理回傳內容。這個專案讓我實際體驗 AI 應用與雲端部署的整合過程。

此外，我曾在 VR 教學專案中實作語音合成模型，使用 GPT-SoVITS 進行語音特徵處理（f0、hubert、mel），進一步累積 AI 模型應用經驗。系統與網路方面，我完成 VMware ESXi 雙層虛擬化架構建置，並部署 Windows Server，設定 Active Directory 與正反向 DNS。在網路實作方面，使用 Cisco Packet Tracer 進行 VLAN、Trunk、Routing、ACL 的模擬設定。

未來希望能朝向雲端架構、網路技術與資安整合方向發展，並強化自動化部署、身份管理與跨平台整合的相關能力，過去在專題與實作過程中，有接觸過 VMware 等虛擬化平台，希望能在實務中累積更多經驗，逐步提升技術能力，為團隊提供穩定的協助與貢獻。
